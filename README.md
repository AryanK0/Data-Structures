# Data Structures and Algorithms

This repository contains C++ implementations of various Data Structures and Algorithms problems. The codebase is organized by assignments, covering a wide range of computer science concepts.

## Structure
The repository is divided into 8 assignment folders. Each folder contains the standard questions as well as the 'Additional Questions' merged seamlessly. Files have been descriptively named to reflect the algorithm or data structure they implement (e.g., `Binary_Search.cpp`, `Top_K_Frequent_Elements.cpp`).

## Topics Covered
- **Arrays & Strings:** Manipulation, Sorting, Two Pointers, String Matching.
- **Linked Lists:** Singly, Doubly, and Circular Linked Lists.
- **Stacks & Queues:** Implementations, Infix/Postfix conversions, Min Stack, Circular Queues.
- **Trees & Graphs:** Binary Trees, Binary Search Trees, Heaps.
- **Sorting & Searching:** Selection Sort, Bubble Sort, Binary Search.

## How to Run
All files are standalone C++ programs. You can compile and run them using standard C++ compilers:
```bash
g++ filename.cpp -o output
./output
```
