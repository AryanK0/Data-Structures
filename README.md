# Data Structures and Algorithms in C++

Welcome to my comprehensive repository of Data Structures and Algorithms (DSA) implemented purely in C++. This collection serves as a foundational library of algorithms, data structure implementations, and problem-solving techniques. 

Instead of being divided into assignments, the repository is structured as a flat, highly-accessible library where every implementation is named intuitively based on the problem it solves or the data structure it builds.

## 🚀 Key Topics & Implementations

### 1. Arrays, Strings, & Matrices
The core building blocks of algorithmic problem-solving. This repository contains solutions for array manipulation, string processing, and complex matrix operations (including sparse matrices).
- **Key Files**: `Binary_Search.cpp`, `Matrix_to_Sparse_Form.cpp`, `Top_K_Frequent_Elements.cpp`, `Missing_Number_Binary_Search.cpp`

### 2. Linked Lists
Extensive implementations of Singly, Doubly, and Circular Linked Lists, including deep manipulations like multi-level flattening and cyclic splitting.
- **Key Files**: `Doubly_LinkedList.cpp`, `Multi_Level_Doubly_LinkedList.cpp`, `Split_Circular_List.cpp`

### 3. Stacks & Queues
Classic LIFO and FIFO data structures, applied to real-world algorithmic problems such as parsing mathematical expressions and tracking temperatures.
- **Key Files**: `Infix_to_Postfix.cpp`, `Postfix_Evaluation.cpp`, `Min_Stack.cpp`, `Daily_Temperatures.cpp`, `Circular_Queue.cpp`

### 4. Trees & Heaps
Non-linear data structures focusing on hierarchical data organization, binary search properties, and priority queue mechanisms.
- **Key Files**: `Binary_Tree.cpp`, `BST_Search_Iterative.cpp`, `Max_Heap_Insert.cpp`

### 5. Sorting Algorithms
Foundational sorting algorithms implemented from scratch to understand algorithmic time complexity and swapping mechanics.
- **Key Files**: `Bubble_Sort.cpp`, `Selection_Sort.cpp`, `Improved_Selection_Sort.cpp`

## 🛠️ How to Compile and Run
Every file in this repository is a standalone C++ program designed to run independently. To execute any file, use standard GCC commands:

```bash
# Compile the program
g++ -o output_name Program_Name.cpp

# Execute the compiled binary
./output_name
```
